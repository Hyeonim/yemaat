공지사항 박스
공지사항 데이터 들어갈곳(5개)
최근 업로드한 공지사항에 대해 시간 순으로 5개 노출
제목, 업로드 시간, 게시글 번호가 화면에 노출
공지사항을 눌렀을때 해당 공지사항 상세보기 페이지 전환
랜덤 박스
랜덤 음식 결정 버튼 누르면 사진과 음식이름 출력
영역을 클릭했을 때 음식에 대한 검색 결과 페이지로 전환
새로 추가된 가게 박스
가게 등록일을 기준으로 최근 시간 순으로 5개 노출
새로 추가된 가게는 가게명, 업태가 화면에 노출
추천(슬라이드 형식)
가게예약, 좋아요 많은 순
전체 가게 중에서 예약이 많고, 좋아요를 많이 받은 순(10개 정도)
가게 클릭 시 해당 가게 상세보기 페이지로 전환
이전, 다음 버튼 필요함
